# Horror

Демонстрация ООП на Python — классы животных с наследованием.

## Цель проекта
Показать понимание наследования и полиморфизма на примере классов животных.

## Задачи
- Создать базовый класс Animal
- Реализовать классы-наследники (Dog, Cat и т.д.)
- Переопределить методы в каждом классе

## Результат
Иерархия классов животных с общим интерфейсом и различной реализацией методов.

## Стек технологий
- Python (чистый ООП)

## Запуск
```bash
python main.py
```
